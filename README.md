# 🚗 Bluetooth Controlled Car

A smartphone-controlled robotic car built using Arduino UNO, HC-05 Bluetooth Module, and L298N Motor Driver.

## 📌 Overview

This project allows a user to control a robot car wirelessly using a Bluetooth-enabled Android application.

## 🚀 Features

- Forward Movement
- Backward Movement
- Left Turn
- Right Turn
- Stop Function
- Wireless Bluetooth Communication

## 🛠 Components Used

- Arduino UNO
- HC-05 Bluetooth Module
- L298N Motor Driver
- DC Motors
- Robot Chassis
- Battery Pack

## 📱 Mobile App

Any Bluetooth Terminal App can be used.

![Circuit Diagram](Circuit_Diagram/bluetooth_car_circuit.png)

## ⚙ Working

1. Mobile app sends commands.
2. HC-05 receives commands.
3. Arduino processes commands.
4. L298N controls motors.
5. Car moves according to user input.
